API

服务端接口

当前后端对客户端开放学校、教务导入、版本检查、课程图片识别和题目解析接口。 管理接口需要管理员登录后使用。

Public Endpoints

客户端与调试入口

GET /api/v1/health 服务状态 GET /api/v1/schools 学校列表 GET /api/v1/schools/{schoolId} 学校详情 GET /api/v1/schools/{schoolId}/lesson-times 学校节次时间 GET /api/v1/schools/lesson-time-profiles 通用节次时间
POST /api/v1/imports/{schoolId}/courses 教务课程导入
GET /api/v1/app/version 版本检查
POST /api/v1/app/version/check 版本检查
POST /api/v1/course-pickups 图片识别并生成取件码
GET /api/v1/course-pickups/{code} 按取件码取回 JSON
POST /api/v1/courses/recognize 直接识别课程图片
POST /api/v1/questions/analyze 题目解析
