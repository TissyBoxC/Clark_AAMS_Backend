我的页面

“我的”页面是工具和设置入口，当前包括：

  • 账号入口。
  • 桌面小组件说明入口。
  • 学习通作业辅助。
  • 课程设置。
  • 应用设置。
  • 关于。

顶部个人区域会根据登录状态显示用户信息。未登录时，课程表和桌面小组件仍然可以正常使用。
