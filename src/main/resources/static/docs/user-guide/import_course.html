课程导入

课程导入入口位于“我的”里的“课程设置”，点击右上角菜单后选择“导入课程”。

OCR 图片识别导入

OCR 导入用于从课表图片中识别课程。

使用流程：

  1. 在课程设置页打开“导入课程”。
  2. 选择“使用 OCR 导入”。
  3. 从相册中选择一张或多张课表图片。
  4. 应用上传图片到后端识别。
  5. 识别完成后会生成取件码。
  6. 用户可以复制取件码，也可以继续导入识别出的课程。
  7. 导入前需要设置第一周第一天和每节课开始时间。
  8. 设置课程颜色后确认导入。

注意事项：

  • OCR 功能需要网络连接和后端服务可用。
  • 识别结果依赖图片清晰度和课表格式。
  • 取件码可用于在其他设备或稍后重新获取识别结果。

取件码导入

取件码导入用于通过 8 位取件码获取已识别的课程数据。

使用流程：

  1. 在课程设置页打开“导入课程”。
  2. 选择“使用取件码导入”。
  3. 输入 8 位取件码。
  4. 应用从后端读取对应课程。
  5. 配置第一周第一天和每节课开始时间。
  6. 设置课程颜色后确认导入。

教务系统导入

教务系统导入用于从学校教务系统读取课表。该功能只对已配置教务导入能力的学校开放。

使用流程：

  1. 首次使用时选择支持教务导入的学校。
  2. 在课程设置页打开“导入课程”。
  3. 选择“教务系统导入”。
  4. 应用打开学校教务系统登录页。
  5. 用户在 WebView 中自行登录教务系统。
  6. 应用检测到登录成功后读取登录会话信息。
  7. 后端根据学校适配规则拉取并解析课程。
  8. 用户配置第一周第一天和每节课开始时间。
  9. 设置课程颜色后确认导入。

注意事项：

  • 教务系统导入需要学校配置支持。
  • 登录过程在学校教务系统页面完成。
  • 如果学校登录页、成功跳转规则或教务接口变化，可能需要更新后端适配。

JSON 批量导入

JSON 批量导入用于粘贴课程数组并一次性添加多门课程。

入口：课程设置页右上角菜单，选择“批量添加”。

支持字段包括：

字段 说明
name 课程名称
startWeek 开始周
endWeek 结束周
dayOfWeek 星期，1 到 7 分别表示周一到周日
startLesson 开始节次
endLesson 结束节次
location 上课地点
teacher 教师，选填
rawTime 原始时间文本，选填
weeks 精确上课周，选填，用于单双周或不连续周次
colorValue 课程颜色 ARGB 整数，选填

示例：

[
  {
    "name": "高等数学",
    "startWeek": 1,
    "endWeek": 18,
    "dayOfWeek": 1,
    "startLesson": 1,
    "endLesson": 2,
    "location": "A101",
    "teacher": "张三",
    "weeks": [1, 2, 3, 4]
  }
]

批量添加窗口还支持“导出 JSON”，会把当前课程以应用可重新导入的 JSON 格式复制到剪贴板。
