常见问题

为什么首页周次不对？

请进入“应用设置”，检查“第一周第一天”是否设置为本学期第 1 周的第一天。当前周次由该日期和系统日期计算。

为什么桌面小组件提示先配置上课时间？

小组件需要每节课开始时间来判断下一节课。如果只导入了课程节次，没有配置节次开始时间，小组件无法计算下一节课。请进入“应用设置”配置每节课开始时间。

为什么今天明明有课，小组件显示没有课程？

可能原因包括：

  • 当前日期计算出的周次不在课程周次范围内。
  • 课程的星期设置与今天不一致。
  • 第一周第一天设置错误。
  • 课程使用了精确上课周，但今天所在周不在 weeks 列表中。
  • 桌面小组件刷新延迟，可以打开应用后等待系统刷新。

为什么 ColorOS 里叫“卡片”而不是“小组件”？

这是系统桌面对小组件入口的命名差异。应用侧注册的是 Android 桌面小组件，ColorOS 可能在桌面、卡片中心或插件中心中展示。

重新选择学校会发生什么？

重新选择学校会清空本地课程，并回到学校选择流程。如果需要保留课程，请先在课程设置页使用“批量添加”里的“导出 JSON”复制当前课程数据。

不登录账号能用课表吗？

可以。课程表、课程管理、课程导入和桌面小组件不强制登录。学习通作业辅助需要登录账号。

教务系统导入失败怎么办？

可以尝试：

  1. 确认学校支持教务系统导入。
  2. 确认教务系统账号密码可以正常登录。
  3. 重新进入教务导入页面登录。
  4. 使用 OCR、取件码、JSON 或手动添加作为替代方式。

OCR 识别不准怎么办？

可以尝试：

  1. 使用更清晰的课表截图。
  2. 避免裁切掉周次、星期、节次或课程地点。
  3. 多张图片一起上传。
  4. 识别后手动编辑错误课程。
